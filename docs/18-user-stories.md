# 👥 User Stories

### US-001: Daily Gameplay
- **As a** casual player,
- **I want to** guess the connection between words with progressive clues,
- **So that** I can test my trivia skills in less than a minute every morning.

### US-002: Streak Pride
- **As a** competitive player,
- **I want my** daily solve streak to be saved to my profile,
- **So that** I can show my friends my dedication.
