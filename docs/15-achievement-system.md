# 🏆 Achievement System

Achievements are unlocked automatically during the daily session evaluation and saved locally.

## 🎖️ Unlocked Milestones
- **First Solve:** Unlock badge for completing the first puzzle.
- **Streaker:** Keep a streak active for 7, 30, or 100 days.
- **Speed Runner:** Solve the daily puzzle using only 1 clue.
- **Historian:** Complete 20 puzzles in Archive Mode.
